# Education Knowledge Base

留学知识库 — 公众号文章发布记录 + 留学知识沉淀

## 目录结构

```
├── README.md              ← 本文件
├── articles/              ← 公众号文章发布记录
│   └── YYYY-MM/           ← 按月归档
│       └── {标题}.md      ← 每篇文章
├── knowledge/             ← 留学知识沉淀
│   ├── countries/         ← 各国留学政策与指南
│   ├── universities/      ← 院校信息
│   ├── majors/            ← 专业方向
│   ├── application/       ← 申请攻略
│   └── visa/              ← 签证指南
├── students/              ← 学生档案（脱敏）
└── resources/             ← 原始资料（参考用）
```

## 说明

- **articles/** — 公众号已发布的留学相关文章记录
- **knowledge/** — 系统化的留学知识，按主题分类
- **students/** — 学生咨询记录（脱敏后）
- **resources/** — 收集的参考资料

每次公众号发新文章或积累新知识时，会更新到此仓库。

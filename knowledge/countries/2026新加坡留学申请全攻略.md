## 一、新加坡留学概况

- **学制**：本科3-4年，硕士1-1.5年（授课型），博士4-5年
- **学年**：秋季8月开学为主，部分学校有1月入学
- **费用**：学费约2-6万新元/年，生活费约1.5-2.5万新元/年
- **语言要求**：本科雅思6.0-6.5，硕士6.5-7.0
- **留学优势**：离家近、安全、双语环境、就业前景好、可申请PR

## 三、申请时间线（秋季8月入学）

| 时间 | 任务 |
|------|------|
| 前一年6-9月 | 准备语言考试、选校定位 |
| 前一年9-12月 | 递交申请（NUS/NTU开放） |
| 当年1-3月 | 等待offer，补材料 |
| 当年2-4月 | 确认offer，申请住宿 |
| 当年4-6月 | 申请学生准证(STP) |
| 当年7月 | 入境体检、注册入学 |

## 四、2026年新加坡留学趋势

1. **QS排名集体跃升**：NUS进入全球Top 8，NTU保持Top 30，学历含金量高
2. **中国留学生持续增长**：新加坡已成为中国学生第三大留学目的地（仅次于英美）
3. **就业友好**：毕业后申请就业准证(EP)相对灵活，科技/金融/医疗人才需求大
4. **PR通道**：留学生工作后可申请永久居民，是留学+移民的可行路径
5. **新专业扩张**：2026年NUS/NTU新增多个AI、数据科学、可持续发展相关硕士项目

## 五、签证申请要点（Student's Pass）

- **申请机构**：新加坡移民与关卡局(ICA)
- **申请方式**：通过SOLAR系统在线申请
- **申请时间**：获得offer后立即申请，审理周期约2-4周
- **入学体检**：抵达新加坡后需指定医院体检
- **IPA Letter**：获批后获得原则批准信，持此信入境
- **STP卡**：入境后完成体检和缴费后领取学生准证卡

## 七、打工与就业

- **在读打工**：学期内每周最多16小时，假期无限制
- **毕业就业**：可申请长期访问准证(LTVP)或就业准证(EP)
- **热门就业领域**：金融科技、数据分析、医疗健康、人工智能、供应链
- **薪资水平**：应届毕业生月薪约4,500-6,500新元（科技类更高）

## 📅 申请时间轴

### 大三阶段（2025年上半年）

| 时间 | 任务 |
|------|------|
| 3-5月 | 确定目标院校和专业方向 |
| 5-7月 | 准备GRE/GMAT考试 |
| 6-8月 | 参加科研/实习项目，积累背景 |

### 大三暑假（2025年6-8月）

| 任务 | 说明 |
|------|------|
| 语言考试冲刺 | 托福/雅思出分 |
| GRE/GMT出分 | Top10建议GRE 325+/GMAT 700+ |
| 文书素材收集 | 整理实习/科研/项目经历 |
| 选校定校 | 确定8-12所目标院校 |

### 大四上学期（2025年9-12月）

| 时间 | 任务 |
|------|------|
| **9月** | 申请系统开放，开始网申 |
| **10月** | 商科第一轮截止，完成文书初稿 |
| **11月** | Top10建议11月前完成申请 |
| **12月** | 多数理工/文科专业截止 |

### 大四下学期（2026年1-4月）

| 时间 | 任务 |
|------|------|
| **1月** | 部分专业截止 |
| **2-3月** | 等待offer，参加面试 |
| **3-4月** | 部分专业可持续到3-4月截止 |
| **4月** | 确认录取，准备签证 |

## 🎯 热门专业申请特点

### 计算机科学（CS）

| 项目 | 详情 |
|------|------|
| 截止时间 | 多数12月-次年1月 |
| GRE要求 | Top30建议325+ |
| 托福要求 | 100+（单项20+） |
| 特殊要求 | 部分有视频面试 |
| 热门方向 | AI/ML、数据科学、网络安全 |

### 商科（金融/会计/管理）

| 项目 | 详情 |
|------|------|
| 录取方式 | **分轮次录取** |
| 第一轮 | 9-10月（建议赶此轮） |
| 第二轮 | 12月-次年1月 |
| 第三轮 | 次年3-4月 |
| GMAT要求 | Top30建议700+ |
| 托福要求 | 100+ |

### 工程类

| 项目 | 详情 |
|------|------|
| 截止时间 | 12月-次年1月 |
| GRE要求 | 部分专业不强制 |
| 托福要求 | 80-100 |
| 热门方向 | 电子工程、机械工程、材料 |

### 文科/社科

| 项目 | 详情 |
|------|------|
| 截止时间 | 12月-次年1月 |
| GRE要求 | 多数要求 |
| 托福要求 | 90-100 |
| 特殊要求 | 部分需写作样本 |

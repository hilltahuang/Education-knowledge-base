# 2026美国本科留学全攻略

> 更新日期：2026-04-17 | 来源：College Board + Common Application

---

## 一、美国本科概况

- **学制**：4年
- **申请方式**：Common Application / Coalition Application / 各校独立申请
- **费用**：私立大学5-8万美元/年，公立州外2.5-4.5万美元/年
- **标化**：SAT/ACT（2026年Top 30多数恢复Test-Required）
- **语言**：托福80-100+ / 雅思6.0-7.0+

## 二、申请轮次

| 轮次 | 截止日期 | 约束力 | 录取率 |
|------|---------|--------|--------|
| ED（Early Decision） | 11月1日 | 有约束力（绑定） | 最高 |
| EA（Early Action） | 11月1日 | 无约束力 | 较高 |
| REA/SCEA | 11月1日 | 限制其他EA/ED | 较高 |
| RD（Regular Decision） | 1月1-15日 | 无约束力 | 标准 |
| Rolling | 滚动 | 先到先得 | 因校而异 |

## 三、申请材料清单

1. **申请表**：Common App或学校独立申请
2. **高中成绩单**：GPA + 成绩单（需认证）
3. **标化成绩**：SAT/ACT（多数Top 30强制）
4. **语言成绩**：托福或雅思
5. **文书**：Personal Statement + 补充文书（各校要求不同）
6. **推荐信**：2封教师推荐 + 1封辅导员推荐
7. **活动列表**：课外活动、竞赛、志愿服务等
8. **作品集/面试**：艺术类专业或部分学校要求

## 四、2026年关键变化

1. **标化全面回归**：哈佛、耶鲁、MIT等Top 30恢复SAT/ACT强制提交
2. **托福改版**：1-6分等级制（2026年1月起）
3. **F1签证审查升级**：社交媒体审查、续签必须面签
4. **D/S制度拟取消**：固定停留期（本科4年），超期需申请延期

## 五、费用总览

| 费用项 | 私立大学 | 公立大学（州外） |
|--------|---------|----------------|
| 年学费 | 5-8万美元 | 2.5-4.5万美元 |
| 年生活费 | 1.5-2万美元 | 1-1.5万美元 |
| **4年总费用** | **约180-300万人民币** | **约100-200万人民币** |

## 六、奖学金

- **Need-Blind**：哈佛、耶鲁、普林斯顿、MIT、达特茅斯、阿默斯特学院（国际生同等待遇）
- **Need-Based**：多数私立名校提供，根据家庭经济状况决定
- **Merit-Based**：基于学术/特长的奖学金，公立大学更常见
- **校外奖学金**：Fulbright、各类基金会奖学金

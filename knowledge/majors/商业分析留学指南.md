# 商业分析（BA）留学指南

> 更新日期：2026-04-17

---

## 一、为什么BA是最火交叉专业？

- 商科+数据的交叉方向，就业面宽
- 多数项目有STEM认证（美国可拿36个月OPT）
- 跨申友好：商科/理工科/文科背景均可申请
- 2026fall申请量同比增20%+

## 三、申请要求

| 要求 | 说明 |
|------|------|
| 本科背景 | 商科/CS/数学/工程/经济均可 |
| GPA | 3.3+/4.0 |
| 先修课 | 统计、微积分、编程(Python/R) |
| GRE/GMAT | GMAT 650+ 或 GRE 320+ |
| 语言 | 托福95+ / 雅思7.0 |
| 实习 | 数据分析/咨询/金融行业相关 |

# 留学时间规划总指南

> 更新日期：2026-04-17

---

## 一、硕士申请时间线（以秋季入学为例）

| 时间 | 任务 |
|------|------|
| 前一年1-6月 | 确定留学国家、选校名单、开始准备语言考试 |
| 前一年6-9月 | 考出语言成绩、联系推荐人、开始写文书 |
| 前一年9-10月 | 文书定稿、准备申请材料 |
| 前一年10-12月 | 递交申请（英美首轮） |
| 当年1-3月 | 跟进申请状态、补材料、准备面试 |
| 当年2-4月 | 收到offer、对比选择 |
| 当年4-6月 | 确认入读、申请签证、准备住宿 |
| 当年7-8月 | 签证获批、行前准备、体检 |
| 当年9月 | 入学 |

## 二、本科申请时间线（Common App）

| 时间 | 任务 |
|------|------|
| 高二暑假 | 准备SAT/ACT、开始写PS |
| 高三9月 | Common App开放、确定选校名单 |
| 高三10月 | 完成ED/EA申请 |
| 高三11月1日 | ED/EA截止 |
| 高三12月 | 完成RD申请 |
| 高三1月1-15日 | RD截止 |
| 高三3-4月 | 收到录取结果 |
| 高三5月1日 | 确认入读（National Decision Day） |

## 三、语言考试规划

| 考试 | 建议首考时间 | 建议出分时间 | 刷分次数 |
|------|------------|------------|---------|
| 雅思/托福 | 申请前12个月 | 申请前6个月 | 2-3次 |
| SAT/ACT | 高二上 | 高三上申请前 | 2-3次 |
| GRE/GMAT | 申请前8个月 | 申请前4个月 | 2次 |

## 四、关键原则

1. **越早越好**：所有考试和准备提前1年开始
2. **分批申请**：ED/EA + RD分批递交，提高录取率
3. **留buffer**：每个节点预留2-4周缓冲时间
4. **多国时间不同**：英美截止早，澳港新可晚一些
